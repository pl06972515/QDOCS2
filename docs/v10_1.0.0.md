<br/>

>[!WARNING|style: flat|label: 简要说明 ]
>
><span style='color:RED'>[`JS`中的事件循环`EventLoop`]</span>
>
>- <span style='color:Blue'>[`A`] 调用栈(`Call Stack`)：</span>同步代码执行的地方 (<span style='color:red'> 后进先出`LIFO`)</span>
>
>- <span style='color:red'>[`B`] 宏任务：交由浏览器宿主环境执行异步任务</span> [`setTimeout`]、[`setInterval`]、[`Ajax`]、[`Events`用户交互事件 ]
>
>   <span style='color:Blue'>[ 宏任务队列(`Task Queue`)：存储宏任务回调函数 ]</span>
>
>- <span style='color:red'>[`C`] 微任务：由`JS`引擎环境执行的异步代码</span> [`Promise.then`]、[`Promise.catch`]、[`Promise.finally`]
>
>   <span style='color:Blue'>[ 微任务队列(`Microtask Queue`)：存储微任务回调函数 ]</span> 
>
>
>
> 
><br/>
>
>⚠<span style='color:red'>[ 遇到`宏任务 | 微任务`交给宿主环境浏览器 ] [ 某一时刻任务完成 → 将回调函数放入相应的队列中 ]</span>
>
>⚠<span style='color:red'>[ 事件循环 ] ( 1.调用栈 )`script`同步代码执行完成 → ( 2.调用栈空闲 )检查微任务队列( 执行微任务回调 ) →  ( 3.调用栈空闲 )检查宏任务队列( 执行宏任务回调 ) [ 依次循环 ]</span>
>
><br/>

```csharp
// [ 调用栈 ] Hello Begin
// [ 调用栈 ] Hello 任务开始
// [ 调用栈 ] Hello End
// 
// [ 宏任务 ] Hello setTimeout.resolve
// [ 微任务 ] 任务成功: 200

# [ 1 ] 执行 script 脚本事件宏任务 ( 脚本内的同步代码 )
<script>

    # [ 2 ] 执行同步代码 ( 调用栈 )
    console.log("Hello Begin");
    # [ 3 ] 执行同步代码 ( 调用栈 )
    const task = new Promise((resolve, reject) => {

         # [ 3.1 ] 执行同步代码 ( 调用栈 )
         console.log("Hello 任务开始");

         # [ 3.2 ] 宏任务交由浏览器宿主环境执行 ( 浏览器是多线程的 - 非 JS 主线程 )
         # [ setTimeout 执行完成( 1 秒后 ): 将回调函数 - 放入宏队列 ] 
         setTimeout(() => {

              console.log("Hello setTimeout.resolve");
              # [ 触发: 任务回调 ]
              resolve(200);

         }, 1000); 

    });

    # [ 4 ] 执行同步代码 - 注册回调函数 ( 调用栈 )
    # [ 当任务触发回调( resolve ): 将回调函数 - 放入微队列 ]
    task.then(result => {
         
         console.log("任务成功:", result);
    });

    # [ 5 ] 执行同步代码 ( 调用栈 )
    console.log("Hello End");

</script>


```
