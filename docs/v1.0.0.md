>[!WARNING|style: flat|label: 简要说明 ]
>
>- 数值类型包含：
>
>   整数(`int`)
>
>   浮点数(`float`)
>
>   特殊值(`NaN`、`Infinity`、`-Infinity`)
>
>
>```javascript
>let x = 10;
>console.info(typeof x); // "number"
>
>let y = 10.5
>console.info(typeof y); // "number"
>
>let z = NaN;
>console.info(typeof z); // "number"
>
>
>```
>
