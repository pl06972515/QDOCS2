>
>- `bool`是一个内置的数据类型，用于表示布尔值 <span style='color:Blue'>[ 只有两个值`true`和`false`]</span>
>
>```csharp
># [ 底层机制 ] 把基本数据类型 → 包装成引用数据类型 new Boolean(true)
>let x = true;
>let y = false;
>
>
>```
>
>
>
><br/>

```csharp
{
    
      # [ 显示转换 ] 以下为 false ( 其余都为 true )
      Boolean(false)     → false
      Boolean(0)         → false
      Boolean('')        → false
             
      Boolean(NaN)       → false
      Boolean(undefined) → false
      Boolean(null)      → false
      
}

# 特殊：当左侧为 false → 返回右侧数据
const x = (false || 'Default')

 
```
