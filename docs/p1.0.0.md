>- <span style='color:Blue'>[ 变量声明 ]</span>
>
>  <span style='color:red'>[ 注意事项：变量名称区分大小写`Age, age`是两个不同的变量 ]</span>
>
>```javascript
>let x, y
>console.info(x) // undefined
>
>x = 10
>console.info(x) // 10
>x = 20
>console.info(x) // 20
>
>
>```
>
><br/>
>
>- <span style='color:RED'>[ 变量声明`let`] 块内作用域 ( 只在`{ ... }`块内有效 - 禁止重复声明 )</span>
>- <span style='color:RED'>[ 变量声明`const`] 作用域同`let`[ 声明时必须初始化 ]</span>
>
>```csharp
>let x = "张三"
>// let x = "李四" -- 不可重复声明
>
>function M1() {
>
>     if (true) {
>
>          let x = 2, 
>              y = 3
>                 
>          console.log(x)  // 2
>          console.log(y)  // 3
>
>     }
>
>     console.log(x)     // 张三
>     # console.log(y) - Error：ReferenceError: y is not defined
>
>}
>M1()
>console.log(x)  // 张三
>
>
>```
