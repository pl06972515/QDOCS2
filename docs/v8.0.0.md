!> <span style='color:red'>在`JS`中，迭代器类型为`object Generator`[ 可迭代对象`iterable : for`都可以转换为迭代器 ]</span>

```csharp
# [ 生成器函数 ] Generator Function
function* M1() {

     yield 1;
     yield 2;
     yield 3;

}

--------------------------

let items = M1();
console.log(typeof items); // [object Generator]

# [ for...of ] 遍历可迭代对象 iterable
for (let item of items) {
     
    
}


```
