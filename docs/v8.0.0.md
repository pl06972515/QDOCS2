>[!WARNING|style: flat|label: 简要说明 ]
>
>在`JS`中，迭代器类型为`object Generator`[ 可迭代对象`iterable : for`都可以转换为迭代器 ]
>
>- 序列类型：`Array, str`
>
>```csharp
># [ 生成器函数 ] Generator Function
>function* M1() {
>
>      yield 1;
>      yield 2;
>      yield 3;
>
>}
>
>--------------------------
>    
>let items = M1();
>console.log(typeof items); // [object Generator]
>
>for (let item of items) {
>      console.log(item);
>}
>
>
>```
